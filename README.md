# 项目简介

## 核心功能
- 在线展示普洱茶产品
- 产品购买与支付
- 订单跟踪与管理
- 茶文化相关知识分享
- 互动社区交流
- 购物车功能
- 个人中心信息管理
- 管理员登录与权限管理
- 用户管理
- 订单管理
- 茶叶信息管理
- 数据统计
- 茶叶分类管理
- 营销管理
- 角色管理

## 适用场景
- 普洱茶爱好者获取产品信息
- 茶叶零售商在线销售产品
- 茶文化推广与普及
- 用户之间茶艺交流与分享
- 企业内部茶叶销售与库存管理

## 技术特点
- 采用Spring Boot框架构建
- 使用MySQL数据库存储数据
- 前端采用Vue.js技术实现用户界面
- 实现了用户认证与权限控制
- 提供灵活的订单管理系统
